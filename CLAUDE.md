# Project Instructions

This is a proof-of-concept Astro marketing website.

Goal:
Create a simple, SEO-friendly, statically rendered marketing website.

Rules:
- Keep the project Astro-based.
- Do not use React.
- Do not add unnecessary dependencies.
- Do not focus on design yet.
- Main content must be real HTML.
- Keep the structure simple.
- Use plain CSS or minimal Astro styling.
- Every page should have:
  - title
  - meta description
  - exactly one H1
  - logical headings
- The site will later be deployed to Cloudflare Pages.

Validation commands:
- npm run dev
- npm run build
